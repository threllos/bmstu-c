# Тесты для лабораторной работы №1, задачи №2, варианта №0

## Входные данные
1 строка

## Выходные данные
Пары: слово - кол чество раз, которое оно встречается в строке

## Позитивные тесты:
 - 01 - обычный тест;
 - 02 - одно слово несколько раз;
 - 03 - каждое слово уникально.

## Негативные тесты:
 - 01 - пустая строка;
 - 02 - cлово длиной > 16;
 - 03 - строка длиной > 256;
